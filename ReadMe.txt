iCoffee - тестовый JPA-JSF проект

1) Окружение.

- MySql (испльзовалась версия 5.6.29),
- веб-сервер (испльзовался Apache Tomcat версия 9.0.0).


2) Сборка.

- С помощью скрипта "schemaDB.sql" (путь "iCoffee\src\main\resources\db\schemaDB.sql") создайте базу данных для приложения (имеет первоначальное заполнение).
- Укажите актуальные имя пользователя и пароль для доступа к БД (файл настроек - "iCoffee\src\main\resources\hibernate.cfg.xml").
- Выполните сборку проекта скриптом "makeWar.bat". Результат - архив "icoffee.war".


3) Запуск. Полученный на предыдущем шаге архив запустите при помощи веб-сервера. Для Tomcat:
- Поместите архив в каталог "webapps" вашего Tomcat.
- Запустите Tomcat (если ранее он не был запущен) - "Tomcat\bin\startup.bat".
- После этого тестовое приложение станет доступно по адресу "http://localhost:8080/icoffee/"


Текст задания можно прочесть в файле "iCoffee\CoffeeShop JSF Rest call.txt".
